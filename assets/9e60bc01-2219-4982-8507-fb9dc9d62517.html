快速開始

  1. 下載此檔案並轉為 HTML。
  2. 確認標題、段落、清單、表格、程式碼上色是否正確。
  3. 檢查圖片是否能正常載入（外部 URL 與相對路徑）。
  4. 驗證數學式（若轉換器支援 KaTeX/MathJax）：$E=mc^2$ 與 $$ \int_0^{2\pi}\sin(x),dx = 0 $$

提示：如果你的轉換器支援高亮主題，請切換深/淺色主題，檢查程式碼色彩與等寬字體。

程式碼範例

JavaScript

說明：以下程式碼示範基本事件監聽與 DOM 更新，並於段落中解釋其運作。

// app.js
document.addEventListener('DOMContentLoaded', () => {
  const btn = document.querySelector('#helloBtn');
  const output = document.querySelector('#output');

  btn.addEventListener('click', () => {
    const t = new Date().toLocaleString();
    output.textContent = `Hello, Markdown → HTML! (${t})`;
  });
});

這段程式碼會在使用者點擊按鈕時，將目前時間寫入 #output 元素。

Python

以下展示一個簡單的 HTTP 伺服器範例，並在說明文字中描述參數與行為：

# server.py
from http.server import HTTPServer, SimpleHTTPRequestHandler

class MyHandler(SimpleHTTPRequestHandler):
    def do_GET(self):
        if self.path == "/healthz":
            self.send_response(200)
            self.end_headers()
            self.wfile.write(b"ok")
        else:
            super().do_GET()

if __name__ == "__main__":
    HTTPServer(("0.0.0.0", 8080), MyHandler).serve_forever()

啟動後，瀏覽 http://localhost:8080/healthz 應回應 ok。
